# Caveats

- The packet uses adjacent multimetal OER literature, not a paper on this exact six-element plate.
- Reported overpotentials in the literature are not directly interchangeable unless current density, electrolyte, and synthesis/testing protocol match.
- Mechanistic claims about which element is the active site should be treated as tentative unless supported by the benchmark artifacts themselves.
- This benchmark uses a hidden evaluator, so any final claim must still be labeled as retrospective simulation rather than real experimental validation.
- This is a retrospective benchmark, not a live experiment.
- Evaluation is hidden and external.
- Any interpretation should remain artifact-grounded and avoid overclaiming mechanistic certainty.
